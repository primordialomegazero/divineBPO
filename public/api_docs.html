Divine BPO API

v15.0 — Production-Ready | phi-Harmonic Architecture | 90% AI, 10% Human

System

GET/api/metrics

Real-time system metrics. Lambda, phi constant, active modules, ticket counts.

curl http://localhost:8092/api/metrics
→ {"total_tickets":24,"ai_handled":21,"human_escalated":3,"ai_rate":87.5,"modules_active":14,"lambda":0.4812}
GET/api/info

System information. Version, status, phi constant.

curl http://localhost:8092/api/info
→ {"name":"Divine BPO","version":"15.0","phi":1.6180339887498948482,"status":"operational"}
GET/health

Health check endpoint. Returns 200 if all systems operational.

curl http://localhost:8092/health
→ {"status":"phi-operational","lambda":0.4812}

Tickets

POST/api/tickets

Create a new ticket. AI auto-classifies module and processes. ~10% escalate to human.

curl -X POST http://localhost:8092/api/tickets \
  -H "Content-Type: application/json" \
  -d '{"module":"customer_support","description":"Refund for damaged order"}'
→ {"ticket_id":"TKT-...","status":"ai_resolved","response":"I am happy to assist..."}
GET/api/tickets/:id

Get ticket by ID. Returns full ticket details including AI response.

curl http://localhost:8092/api/tickets/TKT-1783380485850448060
→ {"id":"TKT-...","module":"customer_support","status":"ai_resolved","resolution":"..."}

Authentication

POST/api/auth/login

Authenticate and receive session token. Valid for 24 hours.

curl -X POST http://localhost:8092/api/auth/login \
  -H "Content-Type: application/json" \
  -d '{"username":"admin"}'
→ {"token":"phi-sess-...","expires_at":1783472000}
POST/api/auth/validate

Validate a session token.

curl -X POST http://localhost:8092/api/auth/validate \
  -H "Content-Type: application/json" \
  -d '{"token":"phi-sess-..."}'
→ {"valid":true,"user":"admin"}
